# NameGPT

**NameGPT** is a Transformer-based model designed to generate male and female names.<br>
Implemented it using pytorch.

## Hyperparameters
batch_size = 32<br>
block_size = 23<br>
n_embd = 384<br>
n_heads = 6<br>
n_blocks = 6<br>
dropout_ratio = 0.2<br>
lr = 3e-4<br>
max_iters = 5001<br>
eval_interval = 500<br>
eval_iters = 200<br>
pad_token = 57<br>

## Validation
val_loss = 1.51<br>
